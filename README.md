# Selge Adventure Park

## 🏞️ Köprülü Kanyon Macera Parkı

Selge Adventure Park, Antalya'nın eşsiz doğal güzelliği Köprülü Kanyon'da bulunan macera dolu aktiviteler sunan bir turizm merkezidir.

### 🚣‍♂️ Aktivitelerimiz:

- **Rafting:** Köprülü Kanyon'da profesyonel rafting deneyimi
- **Zipline:** Kuş bakışı manzara ile adrenalin dolu uçuş
- **Buggy Safari:** Off-road macera turu
- **Camping:** Doğa ile iç içe kamp deneyimi
- **Restaurant:** Geleneksel lezzetler ve doğa manzarası
- **Kano Kiralama:** Kendi hızınızda keşif
- **Grup Etkinlikleri:** Özel organizasyonlar

### 📍 Konum:
Köprülü Kanyon, Antalya, Türkiye
